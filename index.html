Edifice:CSS & JS the Rails way

Edifice makes your Rails life easier. It's designed to simplify CSS & JS integration in your Rails app by following the principle of DRY, less obtrusive markup. Stop writing unnecessary javascript boilerplate, stop wasting time namespacing view CSS and start enjoying rails again.

View on Github

But really now, what is Edifice?

Edifice is a Rails gem that simplifies the way you manage your CSS and Javascript. Install it by typing gem install edifice

Namespacing CSS

Automatically generate a namespace for each of your views. Use it to fine tune styling on a page-by-page basis without worrying about altering your global names. After all thinking of names is hard, why not just use the view's name? Let edifice do it for you:

Before: 
<div id="user_profile"></div>
After: 
<body class="c_users v_show l_application">

Page-level Javascript

Condensing your sites javascript assets into one master file is a performance requirement these days. It gets pretty confusing when you only want to run certain bits of code on different pages. Edifice solves this problem by enabling view specific JS.

Just add: 
userShow = {
    	    onReady: function() {
    	      alert('The users#show page is ready!');
    	    }
    	  }

The rest will be taken care of.

Site-wide Javascript

Reluctant to separate your JS by views? Build Edifice widgets & traits to unobtrusively hook up standard behaviors for you

Just add: 
<select name="country" data-widget="styled_select"><option></select>
<input type="text" name="zipcode" data-trait="only_allow_numbers"></input>

Avoid form boilerplate

Tired of writing standard code to deal with form errors? Wondering why rails' unobtrusive remote form's don't handle it for you? Let (edifice-forms)[https://github.com/tmeasday/edifice-forms] help you out:

Just add: 
<form data-remote="true" data-form="show_errors">

Easy non-persistent forms

Need a form that's not based on a DB-backed model? Still want to validate fields and write simple controllers? Yep, (edifice-forms)[https://github.com/tmeasday/edifice-forms] can help with that:

Just add: 
class Feedback < Edifice::Forms::FormModel
  attr_accessor :message
  attr_accessor :email
  	    
  validates :email, :presence => true, :format => {:with => /^.+@.+\..+$/}
  validates :message, :presence => true
  	    
  def save
    SelfMailer.feedback(self).deliver
  end
end

class FeedbacksController < ApplicationController
  def create
    respond_with @feedback = Feedback.create params[:feedback]
  end
end
